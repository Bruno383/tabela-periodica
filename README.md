# Tabela Periódica Interativa

## Sobre o Projeto

A Tabela Periódica Interativa é um site desenvolvido com HTML, CSS e Java, criado com o objetivo de auxiliar estudantes no aprendizado da Química de forma prática, visual e interativa.

O sistema apresenta todos os elementos químicos da tabela periódica. Ao clicar em um elemento, o usuário pode visualizar informações detalhadas sobre ele, facilitando o estudo e a compreensão das propriedades químicas.

---

## Funcionalidades

- Visualização completa da tabela periódica;
- Interface interativa e organizada;
- Exibição de informações dos elementos químicos;
- Navegação simples e intuitiva;
- Sistema educativo para apoio aos estudos.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JS

---

  Informações Apresentadas

Cada elemento químico pode apresentar informações como:

- Nome do elemento;
- Símbolo químico;
- Número atômico;
- Massa atômica;
- Categoria do elemento;
- Características principais;
- Aplicações e curiosidades.

---

## Objetivo

O principal objetivo deste projeto é tornar o aprendizado da Química mais acessível, dinâmico e interativo, ajudando estudantes a compreenderem melhor os elementos químicos da tabela periódica.
